# Original 4'x4'

For 150-250 chicks

## Materials

* 1q 4'x8' sheet of plywood
* 2q 8'x2"x4" dimensional lumber
* Drywall screws
* 2q Porcelain Lamp Sockets
* 1q 150 watt bulb
* 1q 250 watt bulb
* Wire for lamps
* Male cord end

## Cut list
* plywood
  * top
    * 1q 4'x4' plywood
  * sides
    * 4q 1'x4' plywood
* 2x4
  * Rip to 2x2 stock
* 2x2
  * legs
    * 4q 1'x2"x2" 
  * cleats
    * 4q 4'x2"x2"

## Assembly

### Sides

* Attach 4'x2"x2" cleats 4" from long side

      +-----------------------------------------+  ^
      |                                         |  4"
      |                                         |  |
      +-----------------------------------------+  v
      |                                         | < cleat
      +-----------------------------------------+
      |                                         |
      |                                         |
      |                                         |
      |                                         |
      +-----------------------------------------+
* Attach 1'x2"x2" legs to left inner side

      +-----------------------------------------+  ^
      |                                         |  4"
      |                                         |  |
      +-----------------------------------------+  v
      |                                         | <- cleat
      +-+---------------------------------------+
      | |                                       |
      | |                                       |
      | |                                       |
      | |                                       |
      +-+---------------------------------------+
      | |
      | | <- Leg
      | |
      +-+

* Attach lamp holders to "front" and "back"

      +-----------------------------------------+  ^
      |                                         |  4"
      |                                         |  |
      +-----------------------------------------+  v
      |                                         | <- cleat
      +-+---------------------------------------+
      | |                                       |
      | |                /-\                    |
      | |                |*| <-- lamp holder    |
      | |                \-/                    |
      +-+---------------------------------------+
      | |
      | | <- Leg
      | |
      +-+

### Box

* Assemble sides with cleats and legs to inside

      +-+-------------------------------------+-+
      | |                                     | |
      +-+-------------------------------------+-+
      | |                                     | |
      | |                                     | |
      | |+                                   +| |
      | || <-- lamp holder                   || |
      | |+                                   +| |
      | |                                     | |
      | |                                     | |
      +-+-------------------------------------+-+
      | |                                     | |
      +-+-------------------------------------+-+

### Finish
* Screw dry wall screw into top plywood to provide lift handle
* Place plywood top on top of cleats
* Add insulating litter if desired.

# 4'x6'x2' 

Higher for turkey poults.  Should hold 250-300 chicks.

## Materials

* 2q 4'x8' sheet of plywood
* 2q 8'x2"x4" dimensional lumber
* Drywall screws
* 2q Porcelain Lamp Sockets
* 1q 150 watt bulb
* 1q 250 watt bulb
* Wire for lamps
* Male cord end

## Cut list
* plywood
  * top
    * 1q 4'x6' plywood
  * sides
    * 2q 2'x6' plywood
    * 2q 2'x4' plywood
* 2x4
  * Rip to 2x2 stock
* 2x2
  * legs
    * 4q 2'x2"x2" 
  * cleats
    * 2q 6'x2"x2"
    * 2q 4'x2"x2"

# 2'x3'x1' 

Smaller for up to 75 chicks

## Materials

* 1q 4'x4' sheet of plywood
* 1q 8'x2"x4" dimensional lumber
* Drywall screws
* 4q Porcelain Lamp Sockets
* 4q 40 watt bulb (May need LED equivalents to provide light, but not too much heat)
* Wire for lamps
* Male cord end

## Cut list
* plywood
  * top
    * 1q 2'x3' plywood
  * sides
    * 2q 1'x3' plywood
    * 2q 1'x2' plywood
* 2x4
  * Rip to 2x2 stock
* 2x2
  * legs
    * 4q 1'x2"x2" 
  * cleats
    * 2q 3'x2"x2"
    * 2q 2'x2"x2"
